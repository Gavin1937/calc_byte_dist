
# Calculate byte distribution of an input folder

loop through all files under input foler recursively and then count how many times each bytes (0x00 ~ 0xff) appears

## usage:

```sh
python3 calc_byte_dist.py /path/to/folder/
```

## dependencies:

* numpy >= 1.24.2
* pandas >= 1.5.3

install with command

```sh
pip3 install -r requirements.txt
```

## testing:

```sh
python3 calc_byte_dist.py ./data/test1/

# or

python3 calc_byte_dist.py ./data/test2/
```
